## Quick Start

```bash
# Clone the repo
git clone https://github.com/danieloleary/PartnerAgents.git
cd PartnerAgents

# Try the CLI (one-shot)
python scripts/partner_agents/cli.py "status Acme"

# Or start interactive chat
python scripts/partner_agents/cli.py

# Or run the web UI
python scripts/partner_agents/web.py
# Open http://localhost:8000
```
